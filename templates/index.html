The kalman filter is a standard filter in many applications where...

  • we have noisy measurements of a system
  • we want to estimate the true state of the system
  • we have a model of how the system evolves over time
  • hidden states that aren't measured directly can provide additional filter characteristics

Typical applications are in aerospace, autonomous driving, GPS and many more


There are several types of kalman filters, of which some are implemented graphically on this website

  • Linear Kalman Filter (KF)
    • 1-dimensional KF e.g. tracking the altitude of a falling object without air resistance
    • 2-dimensional KF e.g. tracking of an object in x-y-plane
  • Extended Kalman Filter (EKF)
    • 1-dimensional EKF e.g. tracking the altitude of a falling object with air resistance
    • 2-dimensional EKF e.g. tracking objects with circual paths (think e.g. cars!)
  • Unscented Kalman Filter (UKF)
    • More advanced nonlinear cases
  • Monte Carlo methods
    • Not a kalman filter but a stochastic filtering technique that is often used today
